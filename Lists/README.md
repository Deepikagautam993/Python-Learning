# 📘 Lists in Python

## 🚀 Overview
This module focuses on Python Lists, one of the most commonly used and powerful data structures.

Lists are used to store multiple items in a single variable and are widely applied in programming and Data Structures & Algorithms (DSA).

## 🎯 Learning Objectives
After completing this module, I learned:
- How to create and use lists in Python
- Understanding indexing in lists
- Accessing elements using positive and negative indexing
- Extracting data using slicing techniques
- Adding and removing elements from lists
- Solving basic list-based coding problems

### 🔹 Creating Lists

- Positive indexing

- List slicing techniques

### 🔹 List Methods
- append()
- insert()
- extend()
- remove()
- pop()
- del
- clear()

## 🧠 Key Concepts
Lists in Python are:
- Ordered collections of elements
- Mutable (can be changed after creation)
- Capable of storing duplicate and mixed data types
