# 🚀 Day-09: Pattern Generator Mini Project

## 📌 Overview

This folder contains my first Python mini project — **Pattern Generator**.

The project combines the concepts learned so far and creates a menu-driven application where users can select different patterns and generate them based on the given size.

---

## 🎯 Project Objective

The main objective of this project is to apply Python fundamentals in a practical program.

The project focuses on:

- Writing reusable code using functions
- Applying loops for pattern generation
- Using conditional statements for decision making
- Taking user input and displaying output

---

## 🧠 Concepts Used

- Functions
- Nested Loops
- For Loop
- Conditional Statements (if-elif-else)
- User Input
- Menu Driven Programming

---

## 📚 Features

✔ User can select different patterns  
✔ User can define pattern size  
✔ Generates patterns dynamically  
✔ Organized code using functions  
✔ Handles invalid choices  

---

## 🔹 Available Patterns

1. Square Pattern  
2. Triangle Pattern  
3. Pyramid Pattern  
4. Diamond Pattern
