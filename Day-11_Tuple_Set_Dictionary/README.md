## 🧠 Key Concepts

### Tuple

- Ordered
- Immutable
- Allows duplicate values

### Set

- Mutable
- Unordered
- Stores only unique values

### Dictionary

- Mutable
- Stores data as key-value pairs
- Keys must be unique
