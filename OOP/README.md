# 🚀 Object-Oriented Programming (OOP) in Python

## 📌 Overview

This module covers **Object-Oriented Programming (OOP)** in Python, one of the most important programming paradigms for building modular, reusable, and scalable applications.

The notebooks in this module are designed with a **beginner-to-advanced**, **interview-oriented**, and **project-based** learning approach. Along with theoretical concepts, this module includes hands-on assignments and real-world projects to strengthen practical understanding.

---

* Inheritance

* Polymorphism

* Method Overloading

* Mini Projects
* Real-World OOP Design
