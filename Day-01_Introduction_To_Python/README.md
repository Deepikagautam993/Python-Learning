# 🐍 What is Python?

Python is a high-level, interpreted and general-purpose programming language.

It is used to give instructions to computers and build different types of applications.

Python is known for its simple syntax and readability.

---

# 👨‍💻 Creator of Python

Python was created by:

**Guido van Rossum**

Python was first released in 1991.

# ✨ Features of Python

## Easy To Learn

Python syntax is simple and beginner friendly.

Example:

```python
print("Hello Python")
```

---

## Readable Language

Python code is easy to understand.

---

## Interpreted Language

Python code is executed using an interpreter.

Execution Flow:

```
Python Code
      |
      ↓
Interpreter
      |
      ↓
Output
```

Python executes instructions line by line.

---

## General Purpose Language

Python can be used for different types of applications.

# ⚙️ Installing Python

To install Python:

1. Visit the official Python website.

2. Download the latest Python version.

3. Install Python on your system.

During installation:

✔ Select "Add Python to PATH"

4. Complete the installation.

---

# 🔍 Checking Python Installation

Open Command Prompt / Terminal.

Run:

```bash
python --version
```

Example output:

```
Python 3.x.x
```

---

# ▶️ Running Python Code

Python code can be executed in two ways.

---

# 1. Interactive Mode

Open terminal:

```bash
python
```

Now write Python code:

```python
print("Hello Python")
```

Output:

```
Hello Python
```

---

# 2. Python File Method

Create a file:

```
hello.py
```

Write code:

```python
print("Hello World")
```

Run:

```bash
python hello.py
```

---

# 📄 Python File Extension

Python files use:

```
.py
```

Example:

```
program.py
```

# 🖨️ print() Function

The print() function displays output on the screen.

Example:

```python
print("Hello")
```

Output:

```
Hello
```

---

# Printing Numbers

```python
print(100)
```

Output:

```
100
```

---

# Printing Calculations

```python
print(10 + 20)
```

Output:

```
30
```

---

# 📝 Comments

Comments are notes written for programmers.

Python ignores comments during execution.

Single line comment:

```python
# This is a comment
```

Multi-line comment:

```python
"""
This is a
multi-line comment
"""
```

---

# ✨ Escape Sequence

Escape sequences format output.

## New Line

```
\n
```

Example:

```python
print("Hello\nPython")
```

Output:

```
Hello
Python
```

---

## Tab Space

```
\t
```

Example:

```python
print("Python\tJava")
```

---

# 📚 Python Syntax Rules

## Case Sensitive

Python treats uppercase and lowercase differently.

Example:

```python
name = "Python"
```

and

```python
Name = "Python"
```

are different.

---

## Indentation

Spaces at the beginning of a line are important.

Example:

```python
if True:
    print("Hello")
```

---

# ❌ Common Errors

## Syntax Error

Occurs when Python rules are broken.

Example:

```python
print("Hello"
```

---

## Name Error

Occurs when a variable is not defined.

Example:

```python
print(age)
```

---

## Indentation Error

Occurs due to incorrect spacing.
