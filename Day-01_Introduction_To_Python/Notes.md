# Day 01 - Introduction To Python Notes 🐍


## 1. What is Python?

Python is a high-level, interpreted and general-purpose programming language.

It is used to give instructions to a computer and create different types of applications.

Python is beginner-friendly because its syntax is simple and readable.


---

## 2. Creator of Python

Python was created by:

Guido van Rossum

Python was first released in 1991.


---

## 3. Why Python is Used?

Python is used in:

### Web Development

Used for creating:
- Websites
- Backend applications
- APIs


### Data Science

Used for:
- Data analysis
- Data visualization
- Working with data


### Artificial Intelligence

Used for:
- Machine Learning
- Deep Learning
- AI applications


### Automation

Used to automate repetitive tasks.


---

## 4. Features of Python

### Easy To Learn

Python has simple syntax.

Example:

print("Hello Python")


### Readable

Python code is easy to understand.


### Interpreted Language

Python code is executed by an interpreter.

Execution:

Python Code
↓
Interpreter
↓
Output


### Cross Platform

Python can run on different operating systems.


---

## 5. Installing Python

Steps:

1. Download Python from official website.

2. Install Python.

3. Check installation using:

python --version


Output:

Python 3.x.x


---

## 6. Running Python Code


### Interactive Mode

Open terminal:

python


Then:

print("Hello")


### File Method

Create file:

hello.py


Run:

python hello.py


---

## 7. Python File Extension

Python files end with:

.py

Example:

program.py


---

## 8. print() Function

print() is used to display output.

Example:

print("Hello")


Output:

Hello


Printing numbers:

print(100)


Calculation:

print(10+20)


Output:

30


---

## 9. Comments

Comments are notes for programmers.

Python ignores comments.


Single line comment:

# This is a comment


Multi-line comment:

"""
This is
a comment
"""


---

## 10. Escape Sequence


### New Line

\n

Example:

print("Hello\nPython")


Output:

Hello
Python



### Tab

\t

Example:

print("Python\tJava")


---

## 11. Python Syntax Rules


### Case Sensitive

Python treats uppercase and lowercase differently.


Example:

name

and

Name

are different.


---

### Indentation

Spaces at the beginning of line are important.


Example:

if True:
    print("Hello")


---

## 12. Common Errors


### Syntax Error

When Python rules are broken.


Example:

print("Hello"


### Name Error

When variable is not defined.


Example:

print(age)


### Indentation Error

Wrong spacing in code.
