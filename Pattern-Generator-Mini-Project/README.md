# 🚀 Pattern Generator Mini Project

## 📌 Overview

This folder contains my first Python mini project — **Pattern Generator**.

The project is a **menu-driven application** that allows users to generate different text-based patterns by selecting a pattern type and providing the desired size. It combines multiple Python concepts learned throughout the fundamentals section into one practical application.

This project is part of my **Python Learning Journey 🚀**, where I build projects to strengthen my programming and problem-solving skills.
