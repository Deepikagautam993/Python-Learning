# 📘 Python Tuple, Set & Dictionary

## 🚀 Overview
This module covers Python’s core built-in data structures:
- Tuple
- Set
- Dictionary

These structures are essential for writing optimized programs and building a strong foundation for Data Structures & Algorithms (DSA).

## 🧠 Key Concepts

### 📌 Tuple
- Ordered collection
- Immutable (cannot be changed)
- Allows duplicate values

### 📌 Set
- Unordered collection
- Stores only unique values
- Mutable data structure

### 📌 Dictionary
- Stores data in key-value pairs
- Keys must be unique
- Highly efficient for searching and mapping
