## 🚀 Key Concepts Learned

- 🔹 Python is a high-level, interpreted, and general-purpose programming language.
- 🔹 Python was created by **Guido van Rossum** and first released in **1991**.
- 🔹 Python programs are executed line by line using the Python Interpreter.
- 🔹 Python files use the `.py` extension.
- 🔹 The `print()` function is used to display output.
- 🔹 Comments improve code readability and documentation.
- 🔹 Escape sequences help format output.
- 🔹 Proper syntax and indentation are essential in Python.
