# 1️⃣ What is Python?

Python is a **high-level**, **interpreted**, and **general-purpose** programming language.

It is used to give instructions to computers and develop a wide variety of applications.

Python is one of the most popular programming languages because of its **simple syntax**, **readability**, and **powerful libraries**.

### ⭐ Key Points

- High-Level Language
- Interpreted Language
- General-Purpose Programming Language
- Easy to Learn
- Beginner Friendly

---

# 2️⃣ Creator of Python

Python was created by **Guido van Rossum**.

- 📅 First Released: **1991**
- 🎯 Goal: Create a simple and readable programming language.

# 4️⃣ Features of Python

## ✅ Easy to Learn

Python has a clean and simple syntax.

```python

```

---

## ✅ Readable Language

Python code looks similar to English, making it easier to understand.

---

## ✅ Interpreted Language

Python executes code **line by line** using an interpreter.

```text
Python Code
      │
      ▼
 Interpreter
      │
      ▼
   Output
```

---

## ✅ Cross-Platform

Python programs can run on:

- Windows
- Linux
- macOS

without changing the code.

## ✅ General-Purpose

# 5️⃣ Installing Python

### Steps

1. Visit the official Python website.
2. Download the latest Python version.
3. Run the installer.
4. ✔ Check **"Add Python to PATH"**.
5. Complete the installation.

### Verify Installation

```bash
python --version
```

Example Output

```text
Python 3.x.x
```

# 9️⃣ Comments

Comments are notes written for programmers.

Python ignores comments during execution.

## Single-Line Comment

```python
# This is a comment
```

---

## Multi-Line Comment

```python
"""
This is a
multi-line comment.
"""
```

---

# 🔟 Escape Sequences

Escape sequences are special characters used to format output.

## New Line (`\n`)

```python
print("Hello\nPython")
```

Output

```text
Hello
Python
```

---

## Tab Space (`\t`)

```python
print("Python\tJava")
```

Output

```text
Python    Java
```

# 1️⃣1️⃣ Python Syntax Rules

## ✅ Case Sensitive

Python treats uppercase and lowercase letters differently.

```python
name = "Python"
Name = "Programming"
```

These are two different variables.

---

## ✅ Indentation

Indentation is mandatory in Python.

```python
if True:
    print("Hello")
```

Incorrect indentation causes an error.

---

# 1️⃣2️⃣ Common Errors

## ❌ Syntax Error

Occurs when Python syntax rules are violated.

Example

```python
print("Hello"
```

---

## ❌ Name Error

Occurs when a variable has not been defined.

```python
print(age)
```

---

## ❌ Indentation Error

Occurs due to incorrect spacing.

```python
if True:
print("Hello")
```
